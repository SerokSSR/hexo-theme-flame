感謝 JerryC 製作的精美主題。

## 新特性

- 基於 `Butterfly 3.0.0 rc1` 版本，~~領先市面上多數魔改~~

- 配置文件與 `Butterfly` 新版本完全兼容，可隨時切換主題

- 代碼框形式、高亮、配色優化，採用 tomorrow 高亮主題

- 頁腳和滾動條樣式增強（Powered by Dreamy.TZK）

- 配置項更廣泛支持 html 書寫

- 簡化配置文件，精簡主題功能，加快網站速度

- 其他細節優化
